# Monitor de Lojas Offline — Standalone

Painel executivo independente para monitoramento em tempo real do status de faturamento das filiais físicas e integração com o e-commerce da VTEX.

## Como Publicar na Nuvem com 1 Clique (Deploy no Render)

Depois de subir este projeto para o seu GitHub, você pode publicá-lo de graça clicando no botão abaixo:

[![Deploy to Render](https://render.com/images/deploy-to-render.svg)](https://render.com/deploy?repo=https://github.com/lukasg64-png/dashboard-diretoria-c/tree/monitor-offline)

> **Nota**: Ao clicar no botão, o Render solicitará as chaves da VTEX (`VTEX_APP_KEY` e `VTEX_APP_TOKEN`) para concluir a publicação.
